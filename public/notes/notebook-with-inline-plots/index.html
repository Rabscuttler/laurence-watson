Notebook with inline plots

Let’s do an inline plot

import numpy as np
import matplotlib.pyplot as plt
%matplotlib inline

x = np.linspace(-np.pi, np.pi, 201)
plt.plot(x, np.sin(x))
plt.xlabel('Angle [rad]')
plt.ylabel('sin(x)')
plt.axis('tight')
plt.show()
